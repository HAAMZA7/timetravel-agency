# TimeTravel Agency

Webapp de demonstration pour une agence de voyage temporel fictive.
Le projet melange landing page narrative, reservation, quiz de recommandation et chatbot conversationnel.

[Voir la demo](https://timetravel-agency-pearl.vercel.app)

## Fonctionnalites

- hero immersif avec animation de fond
- presentation de destinations temporelles
- formulaire de reservation
- quiz de recommandation
- chatbot IA avec fallback local
- interface responsive

## Stack

- HTML
- CSS
- JavaScript
- AOS.js
- OpenRouter API
- Vercel

## Lancer le projet

```bash
git clone https://github.com/HAAMZA7/timetravel-agency.git
cd timetravel-agency
python -m http.server 8080
```

Puis ouvrir `http://localhost:8080`.
